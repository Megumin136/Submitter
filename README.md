**修改密码**  
将默认的密码**changemenow**改为你自己的，默认密码无法提交作业。

使用密码子命令及-s标志后接您的学号。系统将提示输入当前密码（默认为**changemenow**）；然后重复输入新密码两次（输入时内容不会显示——此为安全特性）。新密码长度必须至少为8个字符。

**1️⃣下载笔记本重命名为：**

**2️⃣将文件压缩为ZIP格式：**
dsml2_XX(改为assignment名称)_220252030XXXX(改为自己学号).zip
